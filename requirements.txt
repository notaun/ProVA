# ══════════════════════════════════════════════════════════════════
# .env FILE — required for the email module
# ══════════════════════════════════════════════════════════════════
#
# Create a file named `.env` in the ProVA project root containing:
#
#   GMAIL_ADDRESS=you@gmail.com
#   GMAIL_APP_PASSWORD=xxxx xxxx xxxx xxxx
#
# How to get a Gmail App Password:
#   1. Enable 2-Factor Authentication on your Google account.
#   2. Go to: Google Account → Security → App Passwords.
#   3. Create a password for "Mail" / "Windows Computer".
#   4. Paste the 16-character code into GMAIL_APP_PASSWORD.
#
# The email module will not crash if .env is missing — it will
# prompt you to configure credentials and skip sending.

# TROUBLESHOOTING
# ══════════════════════════════════════════════════════════════════
#
# PyAudio fails to install on Windows
#   pip install pipwin && pipwin install pyaudio
#   Or download a pre-built wheel:
#   https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
#
# python-Levenshtein fails (needs C compiler)
#   Install without it (slower fuzzy matching):
#     pip install fuzzywuzzy
#   Or switch to thefuzz (pure Python, no compiler needed):
#     pip install thefuzz
#
# pywin32 needs a post-install step (run once as Administrator)
#   python -m pywin32_postinstall -install
#
# winsdk install error
#   Comment out the winsdk line — ProVA's toast system has two
#   other fallback layers (PowerShell XML + plyer) built in.
#
# "No working microphone found" at startup
#   · Windows Settings → Sound → Input → confirm mic is listed
#   · Windows Settings → Privacy → Microphone → allow app access
#   · Bluetooth headsets: pair the headset BEFORE starting ProVA
#
# audioop not found (Python 3.13+)
#   pip install audioop-lts
#
# tkinter not found (Linux)
#   sudo apt install python3-tk
#
# zoneinfo not found (Python 3.8)
#   pip install backports.zoneinfo

# Optional noise suppression (improves BT mic clarity, adds ~150ms)
noisereduce>=2.0.1
numpy>=1.24.0   # required by noisereduce
